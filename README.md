# Podcasts Live Website

## [https://nimble-sunshine-caee5c.netlify.app](https://nimble-sunshine-caee5c.netlify.app/).

## Website run command.

```
npm start
```

Or

```
 yarn start
```

## Package Installation Command.

```
npm install
```

Or

```
 yarn
```

## Uses Technology

- React JS
- React router dom
- Tailwind CSS
